Roastify setup for this project:

This is a Node.js + Vite app, so install dependencies with npm, not pip.

1. Install Node.js 20 or newer.
2. Run `npm install` in the project root.
3. In `roastify-backend`, copy `.env.example` to `.env` and set `OPENAI_API_KEY`.
4. Run backend: `cd roastify-backend && npm run dev`.
5. Run frontend: `npm run dev` in project root.

Optional frontend env:
Set `VITE_API_BASE_URL` only if your backend is not running at http://localhost:3001.
