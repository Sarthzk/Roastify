Roastify - Profile Roasting App

This is a Node.js + Vite app that roasts your GitHub, LinkedIn, Instagram, and Twitter/X profiles using AI.

Setup:
1. Install Node.js 20 or newer.
2. Run `npm install` in the project root.
3. In `roastify-backend`, copy `.env.example` to `.env` and set `OPENAI_API_KEY`.
4. Run backend: `cd roastify-backend && npm run dev`.
5. Run frontend: `npm run dev` in project root.

Optional frontend env:
Set `VITE_API_BASE_URL` only if your backend is not running at http://localhost:3001.

Features:
- GitHub: Fully automated using GitHub API
- LinkedIn: Paste URL, share key details (job title, company, experience)
- Instagram: Paste URL, share bio & engagement data
- Twitter/X: Paste URL, describe tweet style & topics
